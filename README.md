# LightServerDemo
Light server demo for occiware tutorial, this project embed a web socket server developed in JavaFX

## How to build
```gradle build```

## How to run 
```gradle run```

This will launch with an empty window, if you run the client, you will see lights appear, turn on and turn off. <br/>
After some will be deleted. <br/>

The main objective is to pilot the occiware tool chains to a final result. This project materialize the final result. <br/>

A tutorial use case is in progress and will be available soon. Stay tuned ! <br/>
